## Hi, I'm Jay 👋

I build small, themed, in-browser tools that turn one keyword into a whole batch of printable puzzles, books, and Instagram reels — no sign-up, everything runs client-side.

### Live apps

| App | Live link | What it does |
| --- | --- | --- |
| 🌀 **Maze book generator** | [maze-generator](https://jayne-07.github.io/maze-generator/) | Type a theme (`animals`, `space`, …) → printable book of 1–500 themed mazes (5×8 / 6×9 / A4 PDF + PNG zip) |
| 🎬 **Maze reels** | [maze-reels](https://jayne-07.github.io/maze-reels/) | 9:16 Instagram reels promoting the maze book — themed maze + mascot walks the solution |
| 🔠 **Word-search book generator** | [wordsearch-generator](https://jayne-07.github.io/wordsearch-generator/) | Type a theme → printable book of themed word-search puzzles with answer-key pages |
| 🎞️ **Word-search reels** | [wordsearch-reels](https://jayne-07.github.io/wordsearch-reels/) | 9:16 Instagram reels promoting the word-search book — grid appears, 3-2-1, highlights sweep |

### Stack

React 18 + Vite + TypeScript, Canvas/MediaRecorder, jsPDF + JSZip, free APIs (Pollinations.ai, Iconify). Every app deploys to GitHub Pages from a single workflow.
