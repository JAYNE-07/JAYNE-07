## Hi, I'm Jay 👋

I build small, themed, in-browser tools that turn one keyword into a whole batch of printable puzzles, books, and Instagram reels — no sign-up, everything runs client-side.

### Live apps — Instagram reels

| App | Live link | What it does |
| --- | --- | --- |
| 🎬 **Maze reels** | [maze-reels](https://jayne-07.github.io/maze-reels/) | Themed maze + mascot walks the solution |
| 🎞️ **Word-search reels** | [wordsearch-reels](https://jayne-07.github.io/wordsearch-reels/) | Grid + clues → 3-2-1 → answers sweep in |
| 🎥 **Crossword reels** | [crossword-reels](https://jayne-07.github.io/crossword-reels/) | Crossword + clues → countdown → letters fill in |
| 📐 **Sudoku reels** | [sudoku-reels](https://jayne-07.github.io/sudoku-reels/) | 9×9 puzzle → countdown → solution fills in |
| 🔄 **Scramble reels** | [scramble-reels](https://jayne-07.github.io/scramble-reels/) | Scrambled words → countdown → letters re-form |

Every PDF book includes an answer key (interleaved or at the back), exports as PDF + PNG zip in 5×8, 6×9, or A4. Every reel is 9:16 1080×1920 MP4, includes procedural audio, downloads as individual files or a single zip.

### Stack

React 18 + Vite + TypeScript, Canvas/MediaRecorder, jsPDF + JSZip, free APIs (Pollinations.ai, Iconify). Every app deploys to GitHub Pages from a single workflow.
